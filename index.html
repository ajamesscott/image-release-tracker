Image Comics Release Tracker

I made this because I wanted to be able to see, at a glance, which new Image trades were coming out every week. It only works a month or two out from the present, past or future, depending on what Image has filled out on their site.

Directions: Click the input to pick a release date. Check the box if you only want to see trades. Hover over a cover to see more details. Check out the repo on Github if you want to contribute, or just see how it works. Please note: does not currently work on mobile / touch devices.
